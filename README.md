# Vision AI 기반 실시간 배리어프리 지도 데이터 엔진

## 프로젝트 개요

본 프로젝트는 가게 입구 이미지에서 계단과 경사로를 탐지하고, 탐지 결과를 바탕으로 휠체어·유모차 이용자의 접근성 등급을 A/B/C로 산출하는 Vision AI 기반 배리어프리 접근성 분석 프로토타입이다. 오픈소스 컴퓨터 비전 모델(YOLO 계열)을 직접 구축한 데이터셋으로 추가 학습(fine-tuning)하여, 사진 한 장만으로 보행 약자의 접근 가능 여부를 자동 판별하는 것을 목표로 한다.

## 탐지 클래스

- `stairs`: 계단
- `ramp`: 경사로

## 접근성 등급 기준

| 계단 탐지 | 경사로 탐지 | 등급 | 의미 |
| --- | --- | --- | --- |
| 없음 | 있음 | A | 휠체어·유모차 접근이 비교적 쉬움 |
| 없음 | 없음 | A | 계단이 없어 접근이 비교적 쉬움 |
| 있음 | 있음 | B | 계단이 있으나 경사로로 우회 가능, 일부 주의 필요 |
| 있음 | 없음 | C | 접근이 어렵거나 보조가 필요함 |

## 개발 흐름

1. Roboflow에서 계단/경사로 이미지 라벨링
2. YOLOv8 형식으로 데이터셋 export
3. Google Colab에서 YOLO 모델 fine-tuning
4. 테스트 이미지 예측
5. 계단/경사로 탐지 결과를 바탕으로 접근성 등급 산출
6. 결과 화면 및 보고서 작성

## 사용 기술

- Roboflow: 이미지 라벨링 및 데이터셋 관리
- YOLO 계열 모델(YOLOv8): 계단/경사로 객체 탐지
- Google Colab: 모델 학습 및 테스트
- Python: 접근성 등급 산출 알고리즘 구현
- GitHub: 코드 및 결과물 관리

## 실행 방법

본 프로젝트는 Google Colab 환경에서 실행한다.

1. `오픈소스프로그래밍 기말과제 팀프로젝트.ipynb`를 Google Colab에서 연다.
2. 필요한 패키지를 설치한다.
3. 노트북의 셀을 순서대로 실행하여 데이터셋 다운로드 → 모델 학습 → 성능 검증을 진행한다.
4. 학습된 모델(`best.pt`)을 불러온 뒤 `analyze_image(이미지경로)` 함수를 실행한다.
5. 실행 결과로 탐지된 객체(계단/경사로), 접근성 등급(A/B/C), 탐지 박스가 표시된 결과 이미지가 출력된다.

### 접근성 등급 산출 로직

```python
has_stairs = "stairs" in detected
has_ramp = "ramp" in detected

if has_stairs:
    grade = "B" if has_ramp else "C"
else:
    grade = "A"
```

## 모델 성능

직접 구축한 데이터셋으로 YOLOv8 모델을 25 epoch 학습한 결과는 다음과 같다. (검증 이미지 78장 기준)

| 클래스 | Precision | Recall | mAP@50 |
| --- | --- | --- | --- |
| 전체(all) | 0.699 | 0.592 | 0.688 |
| stairs (계단) | 0.79 | 0.77 | 0.85 |
| ramp (경사로) | 0.609 | 0.414 | 0.527 |

계단은 형태가 일정하여 높은 탐지 성능(mAP 0.85)을 보인 반면, 경사로는 각도·길이·환경에 따른 형태 변화가 커서 상대적으로 낮은 성능(mAP 0.527)을 보였다. 성능 검증 결과는 `코랩 결과.png`에서 확인할 수 있다.

## 데모 영상

다양한 입력 이미지(계단 → C등급 / 경사로 → A등급 / 계단+경사로 → B등급)에 대한 분석 결과 데모 영상은 보고서 및 제출 자료에 포함되어 있다.

## 한계점 및 향후 계획

- 현재 모델은 계단·경사로의 유무만 판단하며, 경사로의 실제 기울기(각도)는 측정하지 못한다. 향후 SAM 및 깊이 추정 기법을 활용하면 경사 각도까지 반영한 정밀 등급 분류가 가능할 것으로 기대된다.
- 경사로(ramp) 탐지 성능(mAP 0.527)이 계단 대비 낮아, 경사로 학습 데이터의 추가 수집·보강이 필요하다.
- 문턱·출입구 폭 등은 적합한 공개 데이터 확보의 한계로 본 범위에서 제외하였으며, 직접 수집 데이터를 확대하면 탐지 대상에 추가할 수 있다.
